## 一、产品定位

### 1.1 一句话定义

> **把"会学习"这件事，做成可调用的服务。**

### 1.2 产品本质

智学Agent 是面向中国初高中生与大学生的 **AI 驱动学习操作系统**。

它做的事情是：**数字化并整合人类发展至今所有经过科学验证的学习工具，用AI把它们全部打通成一个闭环。**

| 现有工具 | 问题 | 知曜的解法 |
|---|---|---|
| Anki闪卡 | 需手动建卡，孤立于学习活动 | AI自动生成，联动全部模块 |
| 思维导图软件 | 用户手动画，无法与练习联动 | AI自动生成，可交互，与训练闭环 |
| 番茄钟App | 与学习内容完全割裂 | 内置，自动记录学习时长到知识点 |
| 学而思/猿辅导 | 内容封闭，处理不了自己的材料 | AI处理任意材料，或直接生成内容 |
| ChatGPT/Kimi | 万能但无结构，用完即走 | 有完整学习闭环，知识持续积累 |

### 1.3 核心知识来源认知

**AI是主要知识来源**：AI本质上是人类全部知识语料库的对话接口，可直接为任何学科生成结构化学习内容。

```
主入口：用户说"我要学牛顿三定律"→ AI生成笔记+闪卡+测验题+学习计划
次入口：用户上传课件/截图 → AI处理为结构化学习内容
（次入口用于：AI知识盲区 / 课堂专属内容 / 用户个性化定制）
```

### 1.4 产品边界（不做什么）

- 不做课程内容销售（不卖视频课）
- 不做题库购买（AI实时生成）
- 不做硬件（纯软件）
- 不直接给答案（所有答疑走引导流程）
- 核心路径不做语言词汇学习（词汇库作为外语学习可选模块保留）

## 二、目标用户

### 2.1 用户群体

| 群体 | 核心场景 | 核心痛点 |
|---|---|---|
| 初高中生 | 课后消化、备考、提分 | 方法乱、工具散、遗忘快 |
| 大学生 | 课程学习、考研、证书考试 | 自律难、知识宽泛、没有闭环 |

### 2.2 用户的真实痛点

| 表面行为 | 真实问题 |
|---|---|
| 反复重读笔记 | 被动学习，遗忘曲线照常运作，毫无效果 |
| 刷题但错题反复错 | 没有针对性复习，知识点漏洞未被闭合 |
| 用AI直接查答案 | 提取到答案但没有建构理解，下次还是不会 |
| 制定计划但不执行 | 计划不与实际学习状态联动，无人追踪 |
| 笔记很多但考前翻不动 | 笔记是输入，没有经过主动提取和输出的检验 |

```
├─────────────────────────────────────────────────────────────┤
│  第六层：时间管理层     番茄钟 · 每日任务清单 · 考试倒计时      │
├─────────────────────────────────────────────────────────────┤
│  第五层：引导答疑层     苏格拉底引导 · 费曼输出评估            │
├─────────────────────────────────────────────────────────────┤
│  第四层：主动训练层     布鲁姆分层出题 · 交错练习 · 费曼输出    │
├─────────────────────────────────────────────────────────────┤
│  第三层：记忆强化层     AI闪卡 · FSRS间隔重复 · 错题本         │
├─────────────────────────────────────────────────────────────┤
│  第二层：知识加工层     笔记三件套 · 知识点提取 · 知识图谱      │
├─────────────────────────────────────────────────────────────┤
│  第一层：知识来源层     AI主动生成（主）· 用户上传（次）         │
└─────────────────────────────────────────────────────────────┘
                    ↕ 所有层共享
           ┌─────────────────────┐
           │   个人知识库         │
           │  （知识原子池）       │

```

### 数据主流程

```
[用户输入主题 / 上传材料]
         ↓
[知识加工层：生成三件套 + 提取知识点]
         ↓
[知识库：存储知识原子（概念/公式/定理）]
         ↓
    ┌────┴────┐
    ↓         ↓
[闪卡系统]  [出题系统]
[FSRS调度]  [布鲁姆分层]
    ↓         ↓
[错题归档 → FSRS调度复习]
         ↓
[时间管理：今日任务自动生成]
         ↓
[规划洞察：周报 + 薄弱点分析]
```

## 四、功能详述

### 第一层：知识来源层

#### F1.1 AI主动生成内容（主入口）

**用户行为**：输入"我要学[学科][知识点]"

**系统行为**：
1. AI根据布鲁姆分类法判断知识点复杂度
2. 生成结构化知识内容（等同于一份课件）
3. 直接进入第二层加工，输出三件套+知识点+闪卡

**适用场景**：
- 预习新知识点
- 对某个概念不理解想重新学
- 自学任意学科内容

#### F1.2 用户上传材料（次入口）

**支持格式**：图片（JPG/PNG）、PDF、纯文字粘贴

**适用场景**：
- 课堂专属讲义
- AI知识盲区内容
- 用户自定义个性化材料

---

### 第二层：知识加工层

#### F2.1 笔记三件套

对应学习科学：**双重编码 + 康奈尔笔记法**

| 版本 | 对应康奈尔区域 | 功能 |
|---|---|---|
| 精读版 | 笔记区 | 深度理解，含原理/推导/例子 |
| 应试版 | 线索区+总结区 | 考点压缩，降低认知负荷 |
| 知识图谱 | 可视化补充 | Mermaid交互图，双重编码强化 |

**生成方式**：AI主动生成 或 用户上传后AI处理，两条路径输出一致

#### F2.2 知识点自动提取

从内容中提取**知识原子**，写入个人知识库：

```
知识点结构：
├── 名称（如"牛顿第二定律"）
├── 学科（物理）
├── 布鲁姆认知层次（理解/应用/分析...）
├── 内容（定义+公式+例子）
├── 来源（关联笔记ID）
└── 掌握状态（未学/记忆/理解/应用）
```

#### F2.3 难点自动标记

AI识别内容中的高难度点，自动加入"重点练习队列"，优先进入训练层。

---

### 第三层：记忆强化层

对应学习科学：**间隔重复 + 主动提取 + 遗忘曲线对抗**

#### F3.1 AI闪卡系统

- AI从知识点自动生成闪卡（正面：问题/概念，背面：答案/定义）
- 学完笔记后**24小时内**自动推送首次闪卡复习（对抗遗忘曲线）
- 用户可手动新增、编辑闪卡

**闪卡类型**：
- 概念闪卡（什么是X？）
- 公式闪卡（X的计算公式是？）
- 应用闪卡（什么情况下用X？）

#### F3.2 FSRS间隔重复调度

所有闪卡和错题均由FSRS算法驱动复习时间：

| 掌握状态 | 下次复习间隔 |
|---|---|
| 完全不会 | 1天 |
| 模糊 | 3天 |
| 基本会 | 7天 |
| 熟练 | 14天 |
| 掌握 | 30天+ |

答错→间隔重置；答对→间隔延长。

#### F3.3 错题本

答题错误自动归档，三维分类：

```
错题
├── 知识点维度（具体知识原子）
├── 题型维度（选择/计算/应用/证明）
└── 错误原因（粗心/概念不清/方法不会）
```

错题同样进入FSRS调度，优先级高于普通闪卡。

---

### 第四层：主动训练层

对应学习科学：**主动提取 + 布鲁姆分类 + 交错练习**

#### F4.1 AI分层出题

按布鲁姆认知分类法分层出题：

```
记忆层（Remember）：   填空/选择 → "这个公式是什么？"
理解层（Understand）：  简答 → "用自己的话解释这个概念"
应用层（Apply）：       计算/解题 → "用这个公式解下面这道题"
分析层（Analyze）：     对比 → "比较A和B方法的区别"
```

**自适应难度**：根据用户答题结果动态调整出题层次，保持在"最近发展区"（刚好够难）。

#### F4.2 交错练习模式

混合当前知识点 + 历史相关知识点，强迫大脑辨别而非机械重复。

例：学完"牛顿第二定律"后，出题时混入"牛顿第一/第三定律"相关题目。

#### F4.3 费曼输出练习

学完一个知识点后，系统提示：

> "请用最简单的语言，向一个完全不懂的人解释这个概念。"

AI评估解释的准确性，指出理解漏洞，引导补充。

对应费曼学习法：**能教才是真正理解。**

---

### 第五层：引导答疑层

对应学习科学：**苏格拉底法 + 费曼法 + 主动建构**

#### F5.1 苏格拉底式引导

**绝对禁止直接给出完整答案（系统层面强制）**

引导流程：
```
用户提问 → AI追问已知条件 → 引导推导步骤
→ 用户得出结论 → AI给出标准解法对比
最多5轮引导 → 超出给提示词（非完整答案）
```

#### F5.2 知识库联动

答疑时优先基于用户自己的笔记/知识点回答，让学生意识到"自己其实已经学过这个"，强化知识连接。

---

### 第六层：时间管理层

对应学习科学：**注意力管理 + 番茄工作法 + 遗忘曲线时间安排**

#### F6.1 番茄钟

- 内置学习计时器（25分钟专注 + 5分钟休息，可自定义）
- 每个番茄钟结束自动记录学习时长，关联到当前知识点/学科
- 学习数据自动汇入进度仪表盘

#### F6.2 每日任务清单

系统自动生成（用户无需手动规划）：
- 今日到期的闪卡（FSRS调度）
- 今日到期的错题复习
- 建议学习的新知识点（基于学习计划）
- 用户手动添加的任务

#### F6.3 考试倒计时

- 录入考试日期与科目
- 倒计时实时可见
- 自动影响学习计划密度（越近考试，计划越密）

---

### 第七层：规划与洞察层

对应学习科学：**元认知 + 反思性学习**

#### F7.1 AI学习计划生成

输入：目标（考试/掌握某知识点）+ 可用时间
输出：
- 分阶段学习路径（先学什么、再学什么）
- 每日任务分配

#### F7.2 进度仪表盘

- 知识点掌握度地图（哪些会了/哪些还不会，按布鲁姆层次展示）
- 学习时长趋势（番茄钟数据）
- 连续打卡记录（习惯养成）

#### F7.3 薄弱点分析

- 自动识别错误率最高的知识点
- 主动推送强化练习
- 展示"如果现在考试，哪些点最危险"

#### F7.4 周复盘报告（AI自动生成，每周日）

- 本周新学知识点数量
- 闪卡复习完成率
- 最大薄弱点Top3
- 下周建议学习重点

| 自适应难度 | 最近发展区理论（Vygotsky） |
