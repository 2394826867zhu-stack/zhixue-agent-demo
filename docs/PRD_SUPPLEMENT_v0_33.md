## 一、学习闭环数值参数（锁定）

| 项 | 值 | 说明 |
|---|---|---|
| 24h 首次复习 | 已实现 ✓ | Flashcard.due_date = today+1，Celery 每小时扫 |
| 苏格拉底引导上限 | **5 轮** | 第 6 轮自动给"提示词卡片"（关键 hint 但非完整答案） |
| 交错练习比例 | **30%** | 70% 当前 KP + 30% 历史相关 KP，学完 ≥3 节课后启用 |
| 自适应难度 step | 连续 3 题正确升一级 / 连续 2 题错降一级 | bloom 上：remember→understand→apply→analyze |
| 错题 vs 闪卡优先级 | **错题 ×1.5 权重** | 排在每日任务前面 |
| 费曼输出评分维度 | 准确性 40% / 完整性 30% / 清晰度 30% | LLM grade，<60 标 wrong + 引导补充 |
